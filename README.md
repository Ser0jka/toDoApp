# Todo App Frontend (учебный CRUD-шаблон)

Простой фронтенд-проект для обучения REST API на примере задач.

Проект повторяет учебный паттерн из `calculator-frontend`:
- `GET` список сущностей
- `POST` создание
- `PATCH` редактирование
- `DELETE` удаление

## Запуск

```bash
npm install
npm start
```

Приложение будет доступно на `http://localhost:3000`.

## Ожидаемое API

Базовый URL: `http://localhost:8080`

### 1) Получить список задач

`GET /tasks`

Пример ответа:

```json
[
  {
    "id": 1,
    "title": "Сделать ДЗ",
    "completed": false
  }
]
```

### 2) Создать задачу

`POST /tasks`

Пример тела запроса:

```json
{
  "title": "Подготовить урок"
}
```

### 3) Обновить задачу

`PATCH /tasks/{id}`

Пример тела запроса:

```json
{
  "title": "Подготовить урок по REST",
  "completed": true
}
```

### 4) Удалить задачу

`DELETE /tasks/{id}`

## Что можно объяснять ученикам

- связь UI-элементов и HTTP-методов
- хранение состояния формы и списка в React (`useState`)
- загрузка данных при старте (`useEffect`)
- обработка ошибок API
- перезагрузка списка после изменения данных
